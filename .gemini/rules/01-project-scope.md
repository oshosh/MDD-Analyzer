# 프로젝트 목표 (불변)

- **이미지 1:1 구현**: 사용자가 제공한 '이미지 5장(엑셀/시트)'에 나온 표/차트 UI를 구조까지 1:1로 동일하게 구현한다. (자막/설명 문구 제외)
- **데이터 무결성**: 결과 수치는 제공된 엑셀/RAW 데이터와 완전히 일치해야 하며, 오차 0을 원칙으로 한다.
- 지원 자산:
  1. 미국주식/ETF (USD 가격 + USDKRW 환율 반영 KRW 가격)
  2. 한국주식 (KRW 가격)
  3. 국제금 (USD 가격 + USDKRW 환율 반영 KRW 가격)

# 절대 금지 (주관/추측 금지)

- 임의의 UI 개선/재배치/컬럼 추가/삭제 금지.
- “대략/추정/근사치”로 결과값을 만들어내는 행위 금지.
- 데이터 소스 변경으로 결과가 달라지는 변경 금지.
- 투자 의견/추천 종목/해석/코멘트 출력 금지.
- 불명확하면 추측하지 말고, 반드시 제공된 엑셀/스크린샷/RAW 구조로 확정 후 구현.

# 완료(합격) 기준

- 동일 기간/동일 데이터에서
  - MDD
  - 구간별 회복율 표(회복율/조건일/주식개장일)
  - 기간 누적 수익률(USD / KRW)
    가 **엑셀과 100% 동일**해야 한다.
